# Hvastra

<img width="459" height="240" alt="hvastra_logo" src="https://github.com/user-attachments/assets/e789c4d1-368d-41a7-a241-afc44b498b19" />

A toolchain for creating turn-based JRPGs as SNES ROMs.

Hvastra, meaning "Sword" in the Thurlsk language, is meant to represent the process of forging an SNES game from scratch. 
Hvastra is being created with the aim to help make the creation of an SNES JRPG possible for anyone who wishes to.
